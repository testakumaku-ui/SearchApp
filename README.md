# 📚 BookSearchApp

Простое Android-приложение для поиска книг, написанное на Kotlin + Jetpack Compose.


🔎 Поиск книг по названию и автору

📖 Отображение списка результатов

🧠 MVVM архитектура

⚡ UI на Jetpack Compose

📦 Локальные тестовые данные (без API)

📂 Структура проекта

com.example.booksearchapp
│
├── data
│ └── Book.kt
│
├── ui
│ ├── SearchScreen.kt
│ ├── SearchViewModel.kt
│ └── SearchState.kt
│
└── MainActivity.kt
